Requirementsâ€‹

Product Requirements:

    1. Dashboard should support multiple resizable charts
    1. Data can come from different data-sources
    1. Each-chart on the dashboard can have a different data-model, however the model that comes from the server is a generic
    1. You can add same charts multiple times and synchronize them using different parameters:
    • ticker
    • timeframe

Non-product requirements:

    1. Network performance. Although, we expect that we have a good network connect, it would be great to understand how to optimize efficiency
    1. Rendering Stability. When we change the parameters of the data, can we still display a partial chart, while fetching a new chunk of data
    1. Legacy API. We need to implement architecture that would support any underlying API as long as it follows a data-format we establish
