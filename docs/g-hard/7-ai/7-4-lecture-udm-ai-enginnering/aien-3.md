# Sec 7-8 Gen AI  

## GenAI Deep Dive  


## 100, GenAI Intro  
제너레이티브 AI 요약 노트

⸻

1. 제너레이티브 AI란 무엇인가

제너레이티브 AI(Generative AI)는 입력(프롬프트)을 기반으로 새로운 콘텐츠를 생성하는 인공지능 시스템이다.
기존 데이터를 단순 분류하거나 예측하는 것이 아니라, 새로운 텍스트·이미지·코드 등을 만들어낸다는 점이 핵심이다.

정의는 다음과 같이 정리할 수 있다.
	•	입력: 프롬프트(텍스트, 명령 등)
	•	처리: 학습된 대규모 모델을 통한 생성 과정
	•	출력: 새로운 콘텐츠 생성

⸻

2. 대표적인 활용 사례

제너레이티브 AI는 다양한 형태의 콘텐츠 생성에 활용된다.
	•	텍스트 생성
	•	블로그 글
	•	이메일
	•	아이디어 브레인스토밍
	•	이미지 생성
	•	미래 도시 스카이라인
	•	예술 작품
	•	코드 생성
	•	HTML 웹페이지 작성
	•	프로그래밍 코드 자동 생성
	•	음악 작곡 등 기타 창작 영역

핵심은 **창의적 합성(creative synthesis)**이다.

⸻

3. 핵심 개념

제너레이티브 AI의 중심 개념은 다음과 같다.
	•	콘텐츠 생성
	•	패턴 생성
	•	창의적 합성

즉, 기존 데이터를 학습해 그 안의 패턴을 이해하고, 이를 바탕으로 새로운 결과물을 만들어낸다.

⸻

4. 주요 구성 요소

제너레이티브 AI는 여러 핵심 모델을 기반으로 작동한다.
	•	대규모 언어 모델(LLM)
	•	예: GPT 계열
	•	텍스트 생성 중심
	•	확산 모델(Diffusion Model)
	•	이미지 생성 중심
	•	트랜스포머(Transformer) 구조
	•	현대 생성 모델의 핵심 아키텍처
	•	코드 생성 모델
	•	예: Codex

이들은 모두 대규모 데이터로 사전 학습된 **파운데이션 모델(Foundation Model)**이다.

⸻

5. 제너레이티브 AI 아키텍처 개요

전체 흐름은 다음과 같다.
	1.	사용자 프롬프트 입력
	2.	파운데이션 모델 통과
	3.	생성 과정 수행
	4.	최종 결과 출력

이를 파이프라인 형태로 이해할 수 있다.
	•	입력 → 모델 처리 → 생성 → 출력

⸻

6. 주방 비유로 이해하기

제너레이티브 AI를 주방에 비유하면 다음과 같다.
	•	재료 = 데이터
	•	셰프 = 모델
	•	레시피 요청 = 프롬프트
	•	요리 결과 = 생성된 콘텐츠

사용자가 요청(프롬프트)을 하면,
모델(셰프)이 학습한 기술과 지식을 바탕으로 새로운 결과물을 만든다.

모델 내부는 블랙박스처럼 보이지만,
방대한 데이터 학습을 기반으로 결과를 생성한다.

⸻

최종 정리

제너레이티브 AI는
	•	입력 프롬프트를 기반으로
	•	대규모 사전 학습 모델을 통해
	•	새로운 콘텐츠를 생성하는 기술이다.

기존 AI가 예측 중심이었다면,
제너레이티브 AI는 창작 중심 AI라고 이해할 수 있다.
