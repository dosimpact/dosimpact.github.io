51,한화 그랑빌, 석계역

그랑빌 ( 석계역 ) 24B평형 (18.13평, 82Bm²/59.94m² ), 8억

https://docs.google.com/spreadsheets/d/11I-IXKvEzN4YG-TT5HH-EulLDfmOD8ObjJ2inVVMKZc/edit?gid=0#gid=0

집값의 예상 상승률에 따른, 원리금 조기 상환 전월세 전환비용 및 금리계산

월계1동, 그랑빌​

[1] 동부간선도로 지하화 사업 1단계 ( 월릉 IC ~ 대치 IC ), 28년 준공 예정

  • 지하 60m에 왕복 4차선 도로를 만드는 사업임, 50분 거리를 10분으로 교통 개선
  • 진행 : 최초 계획 (09년, 오세훈 시장) → 개발 계획 발표 (15년) → 실시 협약 (23년) → 착공 ✅ (24.10) → 완공 (2029)

[2] 서울원 아이파크 ( 광운대역 부근 역세권 개발 )

  • 철도부지가 상업용으로 개발된다.
  • 아파트민간분양, 2264세대(일반분양 6), 분양가 26억 9,700만 ~ 31억 6,400만,
  • 공급 면적, 59평 평단가 - 4,934만 원 / 전용 면적, 43평 단가 - 6,770만 원

[3] GTX-C 노선 ( 2030년 이후 )

  • GTX-C 노선은 2011년 제2차 국가철도망 계획에 반영
  • 실시계획 승인: 2023년 12월 27일 국토교통부로부터 실시계획이 승인
  • 착공식: 2024년 1월 25일 착공식이 개최
  • 당초 준공 목표: 2028년 말 개통을 목표로 추진
  • 2025년 10월 현재 실질적인 공사(삽을 뜨는 실착공)는 지연 ( 지연 원인: 가장 큰 원인은 총사업비 증액에 대한 민자사업자(현대건설 컨소시엄)와 정부(국토교통부, 기획재정부) 간의 협의 난항 )
  • 현실적인 준공일 : 2030년 이후 (최소 2년 이상 지연) ~

GTA 노선은 A, B, C 가 큰 3개 줄기이며, 이 중심에는 삼성역이 있다. ( 용산, 청량리 )
