

# 딥 드릴

문제점 : 기술, 라이브러리 도입 전 올바른 선택을 위해서는 사실 적절한 지식이 필요하다. 
- AI가 가진 지식의 일부를 나의 지식 쳬계로 순식간에 체화 하는 과정이 필요하다.
- 이 공부방식을 정의하고자 한다. 

기대효과  
- Nest.js 기술을 원래 배우는데는 12시간이 걸렸으나 AI을 활용해서 단시간안에 문서화 하고 Best Practice를 구축한다.
- 지식 데이터 베이스를 구축한다. 
- 이런 컨텍스트를 바탕으로 AI에게 바로 시킬 스킬들도 세팅이 된다.  

방법론 
- Step1. 메타 지식 생성하기  : AI에게 무엇을 모르는지 계속 물어본다.
